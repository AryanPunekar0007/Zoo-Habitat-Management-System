# Zoo Habitat & Wellness Management System

## Overview
A text-based Python application designed to assist small zoos in managing animal habitats, tracking health records, and scheduling feeding times within a constrained budget.

## Features
- Budget-aware resource allocation for habitat management
- Health record tracking for multiple animals
- Feeding schedule management with user-friendly interactive menus
- Persistent data storage using CSV files

## Technologies Used 
- Python 3
- CSV file handling

## How to Run 
1. Clone the repository.
2. Run `zoo_ui.py` to start the application.
3. Follow the on-screen menus to manage habitats, animals, and schedules.
